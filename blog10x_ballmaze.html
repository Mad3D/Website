Ball Maze Game

Balancing the Art & Code: A Developer's Day & Night

By day, I'm constantly striving to improve my skills in Epic Games UE5, focusing on creating realistic materials, environments, lighting, and textures. It's a rewarding challenge! At night, I put on my game dev hat and work on a ball maze game to solidify my coding and Blueprint knowledge.

This personal project lets me delve deeper into Unreal Engine's capabilities. It's a fantastic way to learn and grow as a Technical Artist/Developer! Stay tuned for updates on my journey in both professional and personal Unreal Engine projects.

Stay tuned for more updates! I'll be sharing my progress and development journey as this ball maze rolls towards completion.


For WIP1 and WIP2, I refrained from posting any images because my focus was on establishing the core system. I had to implement the new Input Actions system, but more importantly, I wanted to devise an efficient method to store data and settings. This involved creating a centralized system that all menus could communicate with for data retrieval. Along the way, I delved into understanding Widget UI modular design and other related concepts.

WIP 3

Challenges Conquered: This WIP (Work in Progress) #3 milestone represents some key hurdles overcome. I tackled implementing a random starting position, integrating control settings into the menu UI via the Game Instance, and streamlining respawning and pausing mechanics.

Looking Ahead: Excited to tackle the next stage! In WIP #4, I'll prioritize polishing the menu UI, adding a smooth "flow down" effect to transition the Maze movement beyond a 30-degree angle. Finally, I'll rework the maze design to incorporate rounded corners and enhance the material textures for a more visually appealing experience.

WIP 4

Challenges Conquered: This WIP 4 , Implemented an engaging intro sequence that runs every time the Game Starts.
Redesigned the maze layout for improved gameplay. (plan to refine the Maze modeling latter, smaller holes and add beginning and end Decals.).

Looking Ahead: Planning to implement a lives variable for enhanced gameplay. Exploring efficient communication methods like interfaces or Event Dispatchers. Stay tuned for more updates as we continue this exciting journey!
