Product Configurator

Footwear Product Configurator 

In my free time, I’ve been dedicating my energy and coding skills to building a product configurator. Through this blog, I’ll be sharing some of the challenges I encounter and the solutions I implement along the way.

Although this is a personal project, my goal is to make it available for free once it's finished so everyone can have some fun with it.

You can find the latest updates on the left!

Although the UI is quite simple at the moment, I plan to keep adding new features and improvements. The most noteworthy aspect is that the UI is dynamically generated from a Blueprint.

I took this approach despite the initial additional complexity and research required, but it was definitely worth it. This method eliminates the need to manually link events like button clicks or updates when adding new colors or textures. Instead, everything is added to a table, and the UI is automatically generated, seamlessly passing the information along.
